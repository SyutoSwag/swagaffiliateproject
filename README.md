# Swag Affiliate Project

這是一個使用 React + TypeScript + Vite 開發的聯盟行銷專案。

## 功能特點

- 現代化的用戶界面
- 響應式設計
- 數據統計和可視化
- 新聞資訊展示
- 儀表板功能

## 技術棧

- React
- TypeScript
- Vite
- Tailwind CSS
- ESLint

## 安裝與運行

1. 克隆專案
```bash
git clone https://github.com/SyutoSwag/swagaffiliateproject.git
```

2. 安裝依賴
```bash
npm install
```

3. 啟動開發伺服器
```bash
npm run dev
```

4. 構建生產版本
```bash
npm run build
```

## 專案結構

```
├── public/          # 靜態資源
├── src/             # 源代碼
│   ├── components/  # React 組件
│   ├── App.tsx      # 主應用組件
│   └── main.tsx     # 入口文件
├── index.html       # HTML 模板
└── package.json     # 專案配置
```
